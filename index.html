App | Desing

Project Title

Project Info:

Developed a full-stack web-app; Cafeteria is a food and drinks ordering system. The app is split into two front ends: one for the customer to order their drinks and/or food and one for the Cafeteria/admin to receive and manage orders.

• The front end of the app is built using SASS, tailwind CSS as a CSS extension language, and Webpack as a module bundler.
• The back end is built in Node.js with Express and uses MongoDB with Mongoose as a database.
• The app uses passport.js to authenticate the user.

— 1. Once the user opens the app, see the loader and redirect to the home page.
— 2. As soon as a user click on their menu, and add items to the cart.
— 3. As soon as a user clicks on the cart, the user can place an order with COD & Payment with a card. After successful order is placed, then redirect to order status.
— 4. Finally, Cafeteria gets real-time order update from admins, who can control order status using Socket.io.
